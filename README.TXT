Análise de Evasão de Clientes - Telecom X
1. Sobre o Projeto
Este repositório contém o material completo do projeto de análise de evasão de clientes para a empresa fictícia Telecom X. O objetivo foi realizar uma análise exploratória de dados (EDA) para identificar os principais fatores e perfis associados ao cancelamento de serviços.

A análise serve como base para a criação de estratégias de retenção e para o desenvolvimento de futuros modelos preditivos.

2. Estrutura do Repositório
Este repositório está organizado da seguinte forma:

README.md: Este arquivo, com uma visão geral do projeto e instruções.

CHALLENGER.ipynb: O notebook do Google Colab contendo todo o código em Python utilizado para a extração, tratamento, análise e visualização dos dados.

Relatorio_Analise.pdf: O relatório final detalhado, que apresenta os insights, as conclusões da análise e as recomendações estratégicas para a empresa.

3. Tecnologias Utilizadas
Linguagem: Python

Bibliotecas: Pandas, NumPy, Matplotlib, Seaborn

Ambiente: Google Colaboratory

4. Como Executar o Código
Clone este repositório:

Abra o notebook CHALLENGER.ipynb em um ambiente que suporte Jupyter (como Google Colab, VS Code ou Jupyter Lab).

Execute as células em sequência. A fonte de dados é carregada automaticamente pela internet.
